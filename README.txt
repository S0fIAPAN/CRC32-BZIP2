CRC32 рассчитывается на основе файлов; crc-код невозможно вычислить лишь в двух случаях, когда:
- файл пустой;
- текст в файле содержит символы русского алфавита(в этом случае расчет crc-таблицы происходит неверно,
поскольку каждый такой символ занимает не один, а два байта, поэтому в 32-х разрядном регистре при смещении
битов символа на 24 разряда происходит потеря одного байта)
В остальных случаях crc-код вычисляется корректно.
Параметры расчета CRC32:
Width  : 32
Poly   : 0x04c11db7
Init   : 0xffffffff
RefIn  : false
RefOut : false
XorOut : 0xffffffff
